    • a) {Default Route} to access certain directory & (all files inside it) inside the server files structure =>

      Some practical as next examples :

      1. 1) [Definintion inside the {routes/root.js}] => Define several [Default Routes] to access each file inside the [main root directory] of the node server [views/] [according to Express method ] , inside {routes/root.js} file :

        a- Define a [default route] to access main page [index.html] file WIHTIN (main root directory of server ) :
        // [Default express route definitin of home page or [index.html] file , by assign simple req url shortcut value ] :
        router.get( '/' , (res , req) => { // Displaying handling codes} ) ;

        // [Default express route definition of home page or [index.html] file , by assign simple req url full value ] :
        router.get( '/index.html' , (res , req) => { // Displaying handling codes} ) ; [ => Default express route definitin of home page or [index.html] file ]

        // [Default express route definition of home page or [index.html] file , by assign req url with regex flags ] :
        router.get( '^/$|index(.html)?' , (res , req) => { // Displaying handling codes} ) ;



        b- Define a [default route] to access custom page [page.html] file WITHIN (main root directory of server) :
        // [Default express route definition of another custom page file in the server root ] :
        router.get( '/page(.html)?' , (res , req) => { // Displaying handling codes} ) ;



      2. 2) [Definintion inside the {routes/subdir.js} ] => Define several [Default Routes] to access certain [sub-directory] inside the node server structure [views/subdir/] -> to be created inside the {routes/subdir.js} :

        router.get('^/$|index(.html)?', (req, res) => {
        // [sending a file from the [views/subdir/index.html] folder of the server] => this file will be displayed the current req url , by using several methods :
        res.sendFile(path.join( __dirname , '..', 'views' , 'subdir' , 'index.html') ) ;
        });

            const whiteList = ['https://www.yourdomain.com', 'http://localhost:3500', 'http://127.0.0.1:5500'] ;


          2. b- Define a object of {cors} configutration properties to handle the [whitelist] array to be able to access to the our backend ] :
            const corsOptions = {

            origin: (origin, callback) => {
            // Create a condition of the received origin value (host sender request) :
            if (whiteList.indexOf(origin) !== -1 || !origin) { // if the comming request host value IS existed in previous defined list of the accepted host OR the no request host came at all ]
            callback(null, true);
            } else {
            callback(new Error('Not allowed by Cors '));
            }
            },

            // 2- the second property of cors object of the halding the received request :
            optionSuccessStatus: 200
            }
